这篇文章记录本学期（2026年上半年）传感器原理课设制作过程中的杂七杂八的内容。

本项目名：Omnidirectional Tracking Telemonitoring Orthosis （简称OTTO），全向追踪与远程监护矫形轮椅。

用到的模块：

 - 抽签抽到必须要用的：
   1. 人体存在传感器 （用于监测轮椅周围的障碍物）
   2. 薄膜压力检测传感器 （用于监测用户坐姿）
   3. 五向按键模块 （用于用户控制轮椅移动与靠背角度调节）
   4. 步进电机控制模块 （与步进电机一起，调节轮椅靠背角度）
   5. LCD12864液晶屏 （用于显示轮椅状态）
   6. LoRa通信模块 （用于传输轮椅位置信息给服务器）

 - 拿到抽签结果，结合项目自行添加的模块：
   1. TB6612电机驱动器 （用于驱动轮椅的两个轮子的电机）
   2. GPRS定位模块 （用于定位轮椅位置）
   3. 两条履带+电机模块 （用于驱动轮椅移动）
   4. 蜂鸣器模块 （与人体存在传感器结合，作为检测到障碍物时的提示）
   5. 电池与供电模块 （用于给整套轮椅系统供电）

 - 采用的MCU：
   1. 轮椅部分采用ESP32-S3，以便实现蓝牙遥控功能
   2. 服务器接收端采用Arduino leonardo，以便实现高效数据采集

 - 其他的：
   1. 服务器数据采集端使用rust编写的高性能数据采集与展示系统
   2. 买了个测试假人 （橘雪莉可爱捏）
   3. 外壳计划采用3D打印
   4. PCB计划自行绘制，用嘉立创打板
