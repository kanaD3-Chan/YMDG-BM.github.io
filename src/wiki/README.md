如果说学习笔记是探索途中尚未完工的脚手架——布满尝试的痕迹、探索的岔路、甚至被推翻的旧方案——那么 Wiki 便是从这片工地里生长出来的、经过反复敲打与校验的基石与梁柱。

学习笔记记录的是"过程"。它允许不完整，允许存疑，允许写着写着戛然而止。而 Wiki 记录的是"结论"。它是经过验证的、可检索的、可复用的知识成品。当你在学习笔记中跌跌撞撞走完一段路后，这里便是回头整理行囊、绘制地图的地方。

这里的内容按领域分层组织，每一页都是一张独立的知识卡片——你可以从任意入口进入，通过链接在相关主题间跳跃。
